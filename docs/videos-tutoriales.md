# Tutoriales y Videos

Esta sección contiene referencias a tutoriales en video. Los videos están alojados en YouTube o internamente en tu servidor.

## Cómo Crear Tutoriales

### Opción 1: YouTube (Recomendado)

**Ventajas:**
- Gratis
- Alcance global
- Búsqueda y estadísticas
- Fácil compartir por enlace

**Pasos:**
1. Crea una cuenta Google (si no tienes)
2. Ve a [YouTube.com](https://youtube.com)
3. Haz clic en tu avatar → Crear un canal
4. Graba/sube videos con OBS Studio (gratuito) o similar
5. Organiza en playlists
6. Copia enlace del video y pégalo aquí

### Opción 2: Alojamiento Interno

**Requisitos:**
- Servidor web con video stream (Nginx, Apache)
- Espacio en disco (~1GB por video)

**Pasos:**
1. Graba video en MP4 con resolución 720p o 1080p
2. Sube a carpeta `/media/videos/`
3. Vincula en documentación

### Opción 3: Mux, Vimeo o Similar

Servicios que comprimen y optimizan videos automáticamente.

## Tutoriales Recomendados para Grabar

### 1. Introducción al SGC (5 min)

**Contenido:**
- Qué es el Sistema de Gestión de Calidad
- Módulos principales
- Beneficios

**Grabación:**
- Pantalla completa + narración
- Diapositivas PowerPoint (opcional)

**Formato sugerido:** `SGC_01_Introduccion.mp4`

### 2. Crear una No Conformidad (3-4 min)

**Pasos a mostrar:**
1. Acceso a **No Conformidades**
2. Clic en **+ Nueva NC**
3. Llenar formulario
4. Guardar

**Notas importantes:**
- Mostrar campos obligatorios
- Explicar cada sección
- Mostrar vista previa antes de guardar

### 3. Seguimiento de NC (3-4 min)

**Contenido:**
- Cómo ver estado de una NC
- Cambiar estado
- Agregar comentarios
- Generar Orden de Mejora

### 4. Crear Orden de Mejora (3 min)

**Pasos:**
- Crear desde NC o manualmente
- Asignar responsable
- Definir plazo
- Guardar

### 5. Realizar una Auditoría (5-7 min)

**Contenido:**
- Programar auditoría
- Registrar hallazgos
- Clasificar (Conforme/No Conforme)
- Generar reporte

### 6. Usar Códigos QR (3-4 min)

**Pasos:**
- Generar QR desde NC/OM
- Descargar/Imprimir
- Escanear desde celular
- Ver información en QR

### 7. Generar Reportes (3-4 min)

**Contenido:**
- Acceder a Reportes
- Aplicar filtros
- Interpretar datos
- Descargar PDF/Excel

### 8. Administración de Usuarios (4 min)

**Para administradores:**
- Crear usuarios
- Asignar roles
- Cambiar permisos
- Desactivar usuarios

## Guía Rápida: Grabar con OBS Studio

### Descarga e Instalación

1. Ve a [obsproject.com](https://obsproject.com)
2. Descarga para tu sistema
3. Instala (siguiente → siguiente → listo)

### Configuración Básica

1. **Sources:** Clic en **+** → **Display Capture** (para grabar pantalla)
2. **Audio:** Selecciona micrófono
3. **Settings:** 
   - Output: Carpeta donde guardar videos
   - Video: 1920x1080 (Full HD)
   - FPS: 30 o 60

### Grabar

1. Haz clic en **Start Recording**
2. Navega por SGC mientras grabes
3. Narra lo que estás haciendo
4. Cuando termines: **Stop Recording**
5. Video se guardará automáticamente

### Editar (Opcional)

Si quieres agregar:
- Títulos
- Transiciones
- Zoom
- Recortes

Usa software como **DaVinci Resolve** (gratis).

## Mejores Prácticas

✅ **Hace ✓**
- Grabar en resolución 1080p
- Usar micrófono de buena calidad
- Hablar claro y pausado
- Mostrar el proceso completo
- Poner subtítulos (opcional pero recomendado)
- Organizar en playlists por tema
- Actualizar enlaces cuando haya cambios

❌ **Evita ✗**
- Videos muy largos (>10 min sin cortes)
- Música de fondo molesta
- Clickear demasiado rápido
- Dejar silencios largos
- Pantalla desorganizada
- Grabar con mala iluminación (si usas cámara web)

## Alojamiento de Videos

### YouTube (Recomendado)

**Ventajas:**
- Gratis
- Sin límite de almacenamiento
- Búsqueda integrada
- Estadísticas de visualización
- Compartible por enlace

**Desventajas:**
- Requiere conexión internet
- No es privado si lo dejas público
- Dependencia de YouTube

**URL:** https://www.youtube.com

### Mux

**Ventajas:**
- Streaming de calidad
- Estadísticas avanzadas
- API para integración
- Compresión automática

**Desventajas:**
- Servicio pago
- Requiere cuenta

**URL:** https://mux.com

### Servidor Interno (Nginx)

Si quieres alojar en tu propio servidor:

```nginx
server {
    listen 80;
    server_name videos.tudominio.com;
    
    location /videos/ {
        alias /var/www/videos/;
        mp4;
    }
}
```
