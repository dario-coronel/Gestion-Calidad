# Sistema de Gestión de Calidad (SGC)

Bienvenido al manual de uso del **Sistema de Gestión de Calidad (SGC)** de EDP Agro.

Este sistema permite gestionar de manera centralizada:

- **Normas** - Estándares de calidad y sus puntos asociados
- **No Conformidades (NC)** - Registro y seguimiento de problemas de calidad
- **Órdenes de Mejora (OM)** - Acciones correctivas derivadas de NC
- **Proyectos** - Iniciativas de mejora continua
- **Verificaciones** - Validación de cumplimiento de normas
- **QR** - Códigos QR para trazabilidad y auditoría
- **Reportes** - Análisis y estadísticas de calidad

## Acceso al sistema

- **URL:** https://calidad.edpagro.ddns.net
- **Usuario:** Tu usuario corporativo
- **Contraseña:** Tu contraseña
